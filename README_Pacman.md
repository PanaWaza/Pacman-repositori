# Pacman Console Game (Java)

Un pequeño juego estilo Pacman desarrollado en Java desde cero para
practicar lógica, matrices, ciclos y programación orientada a objetos.

## ¿De qué trata?

El jugador controla a Pacman en un tablero configurable (5x6 o 10x10).\
Debe recolectar premios, evitar trampas y sobrevivir con un número
limitado de vidas.

El juego incluye: - Sistema de puntaje - Vidas - Objetos aleatorios
(fantasmas, premios, paredes) - Historial de partidas - Movimiento con
menú interactivo

## Lo que se practica en este proyecto

-   Uso de matrices (`int[][]`)
-   Generación de valores aleatorios
-   Lógica de control con `switch`, `while`, `do-while`
-   Manejo de `ArrayList`
-   Creación de objetos y constructores
-   Validación de datos ingresados por el usuario

## Nota

Este proyecto fue creado como práctica académica para mejorar
habilidades en Java y fortalecer la lógica de programación.
